# Alura-Store: Análisis de Datos de Tiendas

Este proyecto analiza datos de ventas de varias tiendas utilizando Python y Pandas, generando visualizaciones que ayudan a entender el rendimiento de ventas y las categorías más populares.

## Objetivos
- Cargar y manipular datos en formato CSV
- Analizar ingresos y categorías más vendidas
- Crear visualizaciones claras y efectivas
- Proporcionar recomendaciones basadas en el análisis

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Cómo ejecutar el proyecto
1. Abre el cuaderno en Google Colab
2. Sigue los pasos descritos en el análisis para reproducir los resultados

## Resultados destacados
- Se analizaron los ingresos totales y las categorías más vendidas
- Se generaron gráficos para visualizar tendencias y comparaciones

## Recomendaciones
- Se sugiere enfocar ventas en la Tienda 1, que mostró un alto rendimiento en ingresos y ventas

## Habilidades demostradas
- Análisis de datos con Python y Pandas
- Visualización de datos con Matplotlib
- Interpretación de resultados y recomendaciones basadas en datos
