# Alura-Store
# Análisis de Datos de Tiendas - Alura Store

Este proyecto analiza datos de ventas de varias tiendas utilizando Python y la biblioteca Pandas. Se generan visualizaciones para entender mejor el rendimiento de ventas y las categorías más populares.

## Objetivos
- Cargar y manipular datos CSV.
- Analizar ingresos y categorías más vendidas.
- Crear visualizaciones de datos.
- Proporcionar recomendaciones basadas en el análisis.

## Tecnologías Utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Instrucciones para Ejecutar el Proyecto
Para ejecutar este proyecto, abre el cuaderno en Google Colab y sigue los pasos descritos en el análisis.

## Resultados
Los ingresos totales y las categorías más vendidas fueron analizados, y se generaron gráficos para visualizar estos datos.
